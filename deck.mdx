# DTLS

<Box
  bg='black'
  fontSize={50}
  css={{
    fontWeight:'bold'
  }}
>

- Protocolo de seguridad de transporte de datagramas
- Basado en TLS
- Evolución del SSL
- Canal seguro donde se negocia la criptografía entre cliente-servidor

</Box>

---

# SRTP

<Box
  bg='black'
  fontSize={50}
  css={{
    fontWeight:'bold'
  }}
>

- Secure Real Time Protocol
- Define un perfil RTP
- Usado por ejemplo en el cifrado de llamadas de voz

</Box>

# APIs utilizadas

# Conclusiones

<Box
  bg='black'
  fontSize={40}
  css={{
    fontWeight:'bold'
  }}
>

- Integración en un navegador compatible con la especificación HTML5 sin necesidad de instalar un Software propietario.
- Desde el punto de vista del programador, web-RTC es una completa y sólida base  sobre la que construir un proyecto de comunicación RTC, con las garantías de una API bien documentada y respaldada por importantes entidades tecnológicas como son el W3C, IETF y Google.​
- Además es de código libre en todos sus niveles, a diferencia de otros sistemas como los plugins flash de Adobe, u otros sistemas basados en codecs de audio/video propietarios.​
- El principal inconveniente al que se enfrenta esta API es la limitada compatibilidad con los navegadores web existentes. Si bien es preciso recordar que se trata aún de un proyecto en desarrollo.​

</Box>
